# 甜文规范

## 核心理念

**糖分充足，冲突适度** - 让读者吃糖吃到爽。

---

## 核心要求

### 1. 糖分密度

**推荐配置**：
- 小糖（互动、关心）：每章3-5个
- 中糖（表白、亲密）：每3-5章1个
- 大糖（确认关系、重要时刻）：每10章1个

### 2. 甜蜜时刻设计

#### 类型 1：日常小甜
```
"冷吗？"
她还没回答，外套已经披在肩上。
```

#### 类型 2：暖心细节
```
他记得她不吃香菜。
桌上的菜，一根香菜都没有。
```

#### 类型 3：情话
```
"你知道吗？"
"什么？"
"遇见你之后，我才知道什么叫心动。"
```

### 3. 冲突控制

**允许的冲突**：
- 小误会（1-2章解决）
- 外部阻力（家人、工作）
- 性格磨合

**禁止的冲突**：
- 长时间误会（超过5章）
- 第三者插足（真的喜欢）
- BE倾向剧情

---

## 甜度等级

- 微甜：偶尔撒糖
- 中甜：糖分充足
- 齁甜：全程高糖
- 甜到发腻：慎用
